## 💡 CONSEILS D'UTILISATION

### 1. Personnaliser les Témoignages
Dans `index-v2.html`, section `#temoignages` :
- Modifier les textes
- Changer les noms/métiers
- Ajouter de vrais témoignages clients

### 2. Ajuster les Catégories Galerie
Dans `index-v2.html`, pour chaque image :
```html
<div class="gallery-item" data-category="jantes">
```
Catégories : `jantes`, `cadres`, `pieces`

### 3. Modifier les Stats Hero
Dans `index-v2.html`, section hero :
```html
<span class="stat-number" data-count="200">0+</span>
```
Changer `data-count` pour la valeur finale

### 4. Personnaliser le Typing
Dans `script-v2.js`, ligne ~35 :
```javascript
const words = ['Thermolaquage', 'Restauration', 'Personnalisation'];
```

### 5. Images Avant/Après
Remplacer l'image dans la section `.before-after` :
- Une seule image qui montre avant ET après
- Ou utiliser 2 images différentes

## 🐛 TROUBLESHOOTING

### Les particules ne s'affichent pas
→ Vérifier que le CDN Particles.js est chargé  
→ Vérifier la console pour erreurs JS

### Le typing ne fonctionne pas
→ Vérifier que l'élément `.typing-effect` existe  
→ Vérifier la console pour erreurs

### Les filtres ne marchent pas
→ Vérifier les `data-category` sur les images  
→ Vérifier les `data-filter` sur les boutons

### Le slider avant/après ne bouge pas
→ Vérifier que les 2 images ont la même taille  
→ Tester sur desktop (meilleure exp)
